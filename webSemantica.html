O que é HTML?

O acrônico HTML significa em inglês: Hyper Text Markup Language. Para gente aqui fica: Linguagem de marcação de Hipertexto. Por trás de das palavras Hipertexto e Marcação tem muita história e guardam a real essência da função do HTML, onde a Semântica, que nada mais é do que a organização da informação usando HTML.

O que é CSS?

O cascading Style Sheets (CSS) é uma linguagem utilizada para definir a aparência de documentos que adotam para seu desenvolvimento de linguagens de marcação. O CSS define como serão exibidos os elementos contidos no código de um documento e sua maior vantagem é efetuar a separação entre o formato e conteúdo de um documento.

O que é Web Semântica?

A web semântica é nada mais nada menos, que uma web com toda a sua informação organizada de forma que não somente seres humanos possam entendê-la, mas principalmente máquinas. As máquinas nos ajudarão em tarefas que hoje fazemos manualmente.
